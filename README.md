# Тестування програмних засобів кібербезпеки




## 🛠 Технологічний стек
* **Мова програмування:** JavaScript (Node.js) 
* **Фреймворк для тестування:** Jest 
* **Алгоритм шифрування:** Ed448 
  **CI/CD:** GitHub Actions 



##  Структура проєкту
* `src/lib/crypto-libs.js` — підключення криптографічних бібліотек.
* `tests/crypto.test.js` — unit-тести для перевірки коректності алгоритму.
* `.github/workflows/test.yml` — автоматизований запуск тестів при `push` та `pull request`.
* `.github/workflows/codeql.yml` — статичне тестування безпеки (SAST).

##  Запуск проєкту
1. **Встановлення залежностей:** `npm install`
2. **Запуск тестів:** `npm test`
